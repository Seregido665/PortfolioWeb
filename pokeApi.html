Pokédex Web Interactiva

React - JavaScript - Vite - Node.js - HTML - CSS / Bootstrap

Resumen.

Consiste en una aplicación web cuyo objetivo es la consulta de datos a la PokeAPI.
Se presenta un listado completo de los Pokémon donde se puede acceder a una vista con la información específica de cada uno, además de funcionalidades de búsqueda y filtrado
para mejorar la experiencia de navegación y uso.
También incluye un formulario para crear usuarios y mostrarlos dinámicamente en una lista
con operaciones CRUD básicas en el frontend.
Con este proyecto aplico conceptos del consumo de APIs REST, manejo de estados, renderizado de componentes y gestión de formularios.
