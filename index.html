ChessTeacher Project

Aplicación para aprender y practicar ajedrez contra el motor Stockfish en tiempo real. Permite modificar la dificultad, el tiempo y guardar partidas en la nube vinculadas al usuario para revisarlas en un modo análisis. También permite importar y exportar partidas en formato PGN usado en la notación de movimientos.

Incluye autenticación con JWT, cifrado bcrypt, base de datos en MongoDB, historial de partidas con clasificación de jugadas por calidad, y una barra de ventaja entre otros.

Portal de Cliente

Web para la gestión del área personal de clientes de una aseguradora con una página de inicio y un listado completo de pólizas y sus respectivas renovaciones. con filtrado de número de póliza, estado, importe, rango de fechas, y paginación con JavaScript apoyado por Java.

Está desarrollada con Pug para las plantillas, SASS para los estilos, y Gulp para la compilación. Con una arquitectura modular de componentes reutilizables, renderizado de tarjetas desde un JSON y gestión del mismo con backend en Java. Incluye además soporte para i18n para cambio de idiomas.

PokeApi + CRUD básico

Aplicación web para consultar datos de la PokéAPI, desarrollada con TypeScript para garantizar tipado estático y mayor solidez. Presenta un listado completo de los Pokémon con vista detallada de cada uno, búsquedas y filtrado de datos.

Incluye además la función de crear usuarios con operaciones CRUD básicas, manejo de estados, renderizado de componentes e interfaces hechas para la gestión de formularios.
